Role: UI Developer Intern 
Assignment: Animated Score Reveal & Combo Streak UI -Matiks
Time: 3–4 hours
Framework: React Native (Expo), Reanimated 3

Problem Statement : 
After a multiplayer math duel ends, Matiks needs an immersive post-game score reveal screen. You must build it with production-quality animations.

Requirements : 
 ⁠Score Counter Animation
	•	Animate a number from 0 to the player’s final score (e.g., 2840) 
	•	Numbers should appear to “tick up” with slight overshoot
⁠Combo Streak Badge
	•	Show a badge: 🔥 7 Combo Streak!
	•	Entry: scale from 0 → 1.15 → 1.0 with a bounce
	•	The flame emoji should have a looping pulse (scale + opacity) while visible
⁠Rank Reveal
	•	Player rank (#3 of 1,200) slides up from below the fold with a fade
	•	Stagger this 200ms after the score counter completes
Share Button
	•	A “Share Result” CTA with a shimmer/glint effect that runs on loop
	•	On press: brief scale-down → scale-up haptic-like feedback
Plays on Matiks 

Bonus Challenge : 
Add a confetti particle burst using @shopify/react-native-skia when the score finishes counting. Each particle should have randomized trajectory, rotation, and fade-out.

Constraints : 
Use Reanimated 3 (useSharedValue, useAnimatedStyle, withSpring, withTiming, withSequence, withRepeat)
No third-party animation libraries (no Lottie, no Skia unless bonus)
Must work on Android (ios too if you have mac)
No setState inside animation callbacks,  keep animations on the UI thread

Evaluation Criteria : 
Correct use of Reanimated 3 APIs (UI thread safety)     30%
Animation feel — timing, easing, spring config              25%  
Code structure & readability                                              20%   
iOS + Android parity                                                             15%  
Bonus: Skia canvas particle burst on score reveal          10%
